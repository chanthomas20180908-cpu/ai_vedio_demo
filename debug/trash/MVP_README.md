# MVP测试说明

## 目标
验证从输入文件提取提示词并运行3个I2V模型的完整流程。

## 选择的5个提示词

### 1. 蒸汽喷发式暴怒 (qwen-plus)
- **特点**: 头顶喷发白色蒸汽，夸张愤怒表情
- **效果**: 物理化"气炸了"的网络梗，高视觉冲击力

### 2. 委屈大哭 (deepseek-v3.2)  
- **特点**: 自然真实的大哭表现，无特效
- **效果**: 真实感人，突出眼泪质感，极具共鸣感

### 3. 表情包格式崩坏 (deepseek-v3.2)
- **特点**: 数字故障特效，表情抽搐错位
- **效果**: 赛博朋克风格，表达"心态崩了"的新颖方式

### 4. 夸张大哭泪流成河 (qwen-plus)
- **特点**: 动漫式夸张哭脸，眼泪如水柱喷涌
- **效果**: 卡通化泪水特效，搞笑且充满喜感

### 5. 佛系无奈看破红尘 (deepseek-v3.2)
- **特点**: 表情极度平静，灵魂出窍特效
- **效果**: 黑色幽默，讽刺性地表达"我已经麻了"

## 运行的模型

1. **wan-2.2-i2v-plus** - 阿里通义万相
2. **doubao-seedance-1-lite** - 豆包Seedance Lite版
3. **MiniMax-Hailuo-02** - MiniMax海螺

共生成 **5个提示词 × 3个模型 = 15个视频**

## 参数调整

### 修改测试图片

编辑 `run_mvp_i2v_test.py`:

```python
i_path_test = "你的图片路径"
```

### 只测试部分模型

编辑 `run_mvp_i2v_test.py` 的 `enabled_models`:

```python
enabled_models = [
    "wan-2.2-i2v-plus",  # 取消注释想测试的模型
    # "doubao-seedance-1-lite",
    # "MiniMax-Hailuo-02"
]
```

### 只测试部分提示词

编辑 `run_mvp_i2v_test.py` 的 `test_sets` 列表，注释掉不需要的:

```python
test_sets = [
    {
        "name": "蒸汽喷发式暴怒",
        "image_path": i_path_test,
        "prompt": TEST_I2V_PROMPT_STEAM_RAGE
    },
    # {
    #     "name": "委屈大哭",
    #     ...
    # },
    # 其他提示词...
]
```

## 注意事项

1. 确保已安装所有依赖并配置好API密钥
2. 图片路径必须存在且可访问
3. 网络需要稳定，部分API可能需要代理
4. 视频生成需要时间，请耐心等待
5. 生成失败时会在日志中显示错误信息
